## ✨ What is RepoLens?

RepoLens is an AI-powered repository intelligence platform that transforms raw source code into understandable system knowledge.

Upload a ZIP repository and instantly get:

* 🧠 Architecture understanding
* 🗺️ System maps
* 🔐 Security insights
* 💬 Interactive codebase chat
* 📊 Dependency intelligence

No more spending hours understanding unfamiliar projects.

---

## ⚡ Why RepoLens?

Developers constantly face this problem:

> “I have the code… but I don’t understand the system.”

RepoLens acts like a software engineer that explains the repository for you.
